보안 Biz 인터넷이란?

보안 Biz인터넷은 기업용 인터넷에 필수 보안 기능(백신·유해사이트 차단)을 결합해,
별도 보안 솔루션 도입 없이도 안전하고 안정적인 업무 인터넷 환경을 제공하는 서비스입니다.

인터넷 + PC 보안(백신/유해사이트 차단)을 기본 제공
필요시 방화벽(UTM) 보안(주문형보안 SODA50)까지 선택 결합해 통합 보안 구성 가능
보안 서비스 제공으로 보안 수준 강화 + 비용 절감 효과

어떤 고객들에게 필요한가요?

보안 Biz 인터넷은 보안이 중요하지만
별도 보안 인력/예산/운영 체계가 부담인 사업장에 특히 적합합니다.

어떤 특징이 있나요?

  1. 01.
    PC보안 무료제공
    (백신 + 유해사이트 차단)
    • 바이러스/랜섬웨어/멀웨어/피싱 탐지 및 차단을 제공
    • 유해/악성 사이트 접근 차단으로 감염 경로를 줄임
    • 보안 Biz인터넷은 정보보안(PC) 상품을 무료 제공
  2. 02.
    선택 결합
    (방화벽 보안 통합 구성)
    • 외부 침입 차단(방화벽)
    • VPN 등 안전한 접속 지원
    • URL 차단 등 정책 기반 통제
  3. 03.
    “인터넷 + 보안”을
    한 번에 운영 단순화

    인터넷과 보안을 따로 계약/관리하지 않고,
    하나의 서비스 구성으로 도입할 수 있어
    운영 부담이 줄어듭니다.

무엇이 좋아지나요?

  1. 01.
    보안 강화

    보안 Biz 인터넷만으로 기업 정보보호 수준을
    기본 이상으로 강화하고, 백신 및 유해사이트 차단으로
    감염과 피해 가능성을 선제적으로 줄입니다

    정보보호 강화
  2. 02.
    비용 절감

    기본 PC 보안이 포함되어 별도 솔루션 구매나
    라이선스 비용 부담을 줄이고, 빠른 도입과 가벼운
    유지비로 효율적인 운영이 가능합니다

    별도 구매 불필요
  3. 03.
    안전한 업무 인터넷 환경

    앤드포인트(PC)에서 발생하는 위협을 줄이고,
    필요 시 UTM까지 결합해 사내 네트워크 보안 체계를
    한 단계 업그레이드
