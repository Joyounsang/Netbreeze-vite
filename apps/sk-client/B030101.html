전용회선(프리밴)이란?

인터넷전용회선 프리밴은 고객사 LAN Switch와 SK브로드밴드 LAN Switch를 독자적인
광케이블로 연결하는 인터넷 서비스 입니다.

Ethernet 기반으로 고객이 원하는 속도를 제공하여, QoS(Quality of Service) 제공을 통해
트래픽이 몰리는 상황에서도 보다 안정적인 품질을 보장하는 것이 핵심입니다.

어떤 고객들에게 필요한가요?

  • 인터넷 사용 비중이 높고,
    업무 전반에서 안정적인 네트워크와
    트래픽 관리가 중요한 기업

    벤처기업 대기업

  • 최대 10Gbps급의 다양한
    속도 옵션이 필요하고, 품질을
    일정하게 가져가야하는 사업장

    다양한 속도 옵션

  • 인터넷 사용량이 많고, 다수
    이용자가 동시에 접속해 안정적인
    네트워크와 대역폭 관리가 중요한 기관

    대학교 정부기관

  • 대용량 파일 전송 등으로 업무
    트래픽이 지속적으로 발생하며
    안정적인 네트워크 품질이 중요한 환경

    SaaS/클라우드 그룹웨어 ERP

어떤 특징이 있나요?

  1. 01.
    다양한 회선 속도
    및 L3 스위치 제공

    1Mbps ~ 10Gbps 속도 제공, IP 프레임 그대로 전송해 빠른 처리와 100% 대역폭 활용, 고품질 L3 스위치 (QoS 지원) 제공

  2. 02.
    선로 이중화
    구성 가능

    2포트, 2코어 구축을 통해
    선로 이중화 구성 가능

  3. 03.
    인터넷 회선
    (BGP연동 → 공인IP 유지)

    사업자 이원화로 기존 사용 중인
    공인 IP 변경 없이 인터넷 서비스 제공

  4. 04.
    다이나믹 라우팅
    정책 적용 가능

    Best Path 기반으로 패킷 라우팅 가능한
    다이나믹 라이팅 정책을 적용 (BGP설정은 중형 스위치 사용 시 가능)

무엇이 좋아지나요?

  1. 01.
    업무용 인터넷
    품질 안정화

    QoS 기반으로 품질을 안정적으로 제공해,
    다수 사용자가 동시에 접속하는 환경에서도
    체감 품질 변동을 줄이는 효과가 있습니다.

    안정화
  2. 02.
    빠른 대용량
    데이터 전송

    광 Ethernet 기반으로 고객사까지 고속 전송 환경을
    구성해, 업로드/다운로드 및 내부 시스템 연동이 많은
    조직에 유리합니다.

    빠른 속도
  3. 03.
    장애/이슈 대응 체계와
    업무공백 최소화

    전담 지원 기반으로 즉각적인 장애대응 및
    긴급출동이 가능하고, 24시간 모니터링 서비스로
    운영 리스크를 낮출 수 있습니다.

    관제서비스
